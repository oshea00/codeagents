# Example From [Agentic Coder - Part 2](https://oshea00.github.io/posts/agentic-codegen-part2/)

## Running the example
To build:
```bash
docker build -t test .
```

To run tests:
```bash
docker run -it test
```

To run app:
```bash
docker run -it test python main.py
```

## Notes

The ```agentic_flow_results.json``` shows the context used in the various iterations.
